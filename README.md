## Description

[Nest](https://github.com/nestjs/nest) con uso de netsJS se contruyó un REST API para una Aplicación de notas

## Installatio

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
